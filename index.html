Clojure Finland

About us

We want to bring together the Clojure hackers in Finland to help each other, talk about what we love, promote Clojure and hack interesting stuff.

What is Clojure?

Clojure is a dynamic general-purpose functional programming language that runs in Java Virtual Machine. For more info check out Learn Clojure, Clojure.org and Clojure Wikipedia.
